# Brand and Product Logos

All organizational brand and product logos are available for consumption from
this resource.

## Brand Logos

Depending on your use case, our brand and product logos can be consumed by dependent projects in a couple of different ways. The logo images are available as standalone SVG image files, or, we also have [a logo component available](/?path=/story/components-logo--brand-logo) as well, which provides some flexible options for rendering the logo images.

### Horizontal Variant
To import and use one of the horizontal brand logo SVGs directly, you can import or reference it from this project in a manner similar to this, depending on your use case:

```js
import AOEULogo from "@aoeu/ui-common/images/logos/aoeu-logo-horizontal.svg"
```

This can then be used as the source for an image tag in the following manner:

```jsx
<img src={AOEULogo} alt="The Art of Education University Logo - horizontal" />
```

They are also available wrapped in a component as demonstrated below, and can be imported as:

```js
import { AOEULogo } from "@aoeu/ui-common/components/logos"
```

  size="xs"
  className={styles.logo} />
